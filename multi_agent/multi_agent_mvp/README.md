# Multi-Agent Operations Automation System (MVP)

## 🚀 Overview
This is a production-style MVP of a multi-agent system designed for automated content operations.

## 🧠 Architecture
- Planner Agent
- Research Agent
- Content Agent
- Optimize Agent
- Publisher Agent
- Orchestrator (Pipeline Controller)

## ⚙️ Features
- Multi-agent collaboration
- Pipeline orchestration
- Memory logging
- Batch task processing
- Easily extendable

## ▶️ Run
```bash
pip install -r requirements.txt
python main.py
```
